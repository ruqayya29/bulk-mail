📧 Bulk Mail App
A full-stack web application that allows you to send bulk emails to multiple recipients at once by uploading an Excel file containing email addresses.

🚀 Features

Upload an Excel (.xlsx) file containing email addresses
Compose a custom email message
Send emails to all recipients in one click
Real-time sending status indicator
Email credentials stored securely in MongoDB


🛠️ Tech Stack
##Frontend

React.js
Tailwind CSS
Axios
SheetJS (xlsx)

##Backend

Node.js
Express.js
Nodemailer
Mongoose (MongoDB)
